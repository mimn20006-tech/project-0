إعداد تسجيل الدخول بجوجل

  1. ادخل Google Cloud Console وأنشئ مشروع جديد.
  2. فعّل Google Identity Services أو OAuth Consent Screen.
  3. أنشئ OAuth Client ID (نوع Web).
  4. أضف Authorized Redirect URI: https://hand-aura-production.up.railway.app/api/oauth/google/callback
  5. ضع القيم في متغيرات البيئة: 
    GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
BASE_URL=https://hand-aura-production.up.railway.app

إعداد تسجيل الدخول بآبل

Apple لا يعمل على localhost — تحتاج دومين و HTTPS.

  1. سجّل في Apple Developer.
  2. أنشئ Service ID وأضف Redirect: https://your-domain.com/api/oauth/apple/callback
  3. أنشئ Key (AuthKey.p8) وسجّل Team ID وKey ID.
  4. ضع القيم في متغيرات البيئة: 
    APPLE_CLIENT_ID=...
APPLE_TEAM_ID=...
APPLE_KEY_ID=...
APPLE_PRIVATE_KEY_FILE=C:\path\AuthKey.p8
BASE_URL=https://your-domain.com
